## About the Project

A modern, full-stack **E-Commerce Web Application** built with the **MERN Stack**, offering a smooth shopping experience — from browsing products to secure checkout.

Comes with a **fully functional Admin Panel** to manage products, orders, and users. The entire app is **Dockerized** with a **CI/CD pipeline** for automated builds and deployment — making it fully production-ready. 🚀

## Project Structure

```
ecommerce-app/
│
├── 📂 frontend/                   # React + Vite App
│   ├── src/
│   │   ├── components/            # Reusable UI components
│   │   ├── pages/                 # Route-level pages
│   │   ├── context/               # Context API state
│   │   └── hooks/                 # Custom React hooks
│   └── Dockerfile                 # Frontend Docker config
│
├── 📂 backend/                    # Node.js + Express API
│   ├── controllers/               # Route logic
│   ├── models/                    # MongoDB schemas
│   ├── routes/                    # API endpoints
│   ├── middleware/                 # Auth & validation
│   └── Dockerfile                 # Backend Docker config
│
├── 📂 admin/                      # Admin Panel
│   └── ...
│
├── docker-compose.yml             # Multi-container setup
└── .github/
    └── workflows/                 # CI/CD GitHub Actions
```
